External Resource Mapping & Reference Directory

An official cross-reference directory mapping validated portals, dataset trackers, and verification architectures across our extended network platforms.

  • Research Plymouth Portal Root https://www.researchplymouth.com/

    Primary entryway for academic tracking indices, housing baseline metric infrastructures and core structural updates.

  • Academic Journal Guide (AJG) Data Matrix https://www.researchplymouth.com/ajg/

    Interactive search index containing journal rankings, model scores, and evaluation matrices for business administration and entrepreneurship publications.

  • ARWU Shanghai Ranking Performance Index https://www.researchplymouth.com/arwu/

    Searchable institutional directory tracking the Academic Ranking of World Universities (ARWU), filtering performance score aggregates and global standings.

  • Systematic Literature Review (SLR) Reference Matrix https://www.researchplymouth.com/slr/datatables/

    Central query engine compiling systematic literature review extractions, variable groupings, and research synthesis reporting dashboards.
